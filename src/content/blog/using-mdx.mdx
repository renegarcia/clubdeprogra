__MDX__ es un tipo especial de Markdown que soporta sintaxis de JavaScript y JSX. De esta manera, los archivos Markdown pueden incorporar 
[JavaScript y componentes de interfaz de usuario](https://docs.astro.build/en/guides/markdown-content/#mdx-features) para realizar tareas como crear gráficas interactivas o alertas.

## Ejemplo

Para incorporar un componente MDX se utliza código parecido al siguiente: 

```astro
import HeaderLink from '@/path/to/my/components/HeaderLink.astro';

<HeaderLink href="#" onclick="alert('clicked!')">
	Embedded component in MDX
</HeaderLink>
```

El resulado es una página dinámica, parecido a lo siguiente, deberías ver un vínculo interactivo en el párrafo siguiente,

import HeaderLink from '../../components/HeaderLink.astro';

<HeaderLink href="" onclick="alert('clicked!')">
	Embedded component in MDX
</HeaderLink>
